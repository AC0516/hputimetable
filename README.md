# hputimetable

怪兽课表是一款免费、开源的通用型课表软件，可以无缝从超级课程表和课程格子（暂不支持）的账户以及课程码中导入数据，并且有桌面小部件和学校专区，快来体验吧，感觉好用就分享给你的朋友吧~

涉及到的技术点：

- ButterKnife : 使用注解进行View绑定
- LitePal : SQLite 数据库的简化
- Bugly : 腾讯的管理BUG的平台
- Retrofit : 网络请求库
