=======
hpu小课:一款精致、简洁的课表软件.

它是为河南理工大学学生设计的一个蹭课软件,但你可以通过本小项目的学习进而完成自己的课程表软件.

涉及到的技术点：

- ButterKnife : 使用注解进行View绑定
- LitePal : SQLite 数据库的简化
- Bugly : 腾讯的管理BUG的平台
- Retrofit : 网络请求库
